filemaker.tmbundle by DonovanChan

TextMate Support for FileMaker Pro

This is a bundle for use in TextMate (text editor for Mac OS). It adds functionality that supports development of FileMaker Pro databases.

Sublime Text 2 users, see the SublimeText2 branch. Note, the Sublime Text 2 version currently only supports snippets and syntax highlighting.

Become a FileMaker ninja with these tools in your belt!

  • Tab triggers for FileMaker functions
  • Documentation for functions, script steps and error codes
  • Commands
    • Manipulating/generating calculations
    • Manipulating/generating FileMaker clipboard XML
    • Extracting data from the DDR
    • Extracting data from import.log files
  • Syntax highlighting
    • FileMaker
    • FileMaker Clipboard
    • FileMaker Log
    • FileMaker Hash (Generated using #( ) custom function)
  • Code folding
  • Help section
  • Full documention on most commands (displayed when running command with empty input)

Dependencies

  • TextMate (Available on OS X only. 30-day trial available)
  • Ruby 1.8.7 or newer (should be installed by default on OS X)
  • FileMaker Pro or FileMaker Pro Advanced

Installation

Simple Method

See the installation video.

  1. Install TextMate
  2. Download the files. (zip or tar formats available)
  3. Uncompress/extract the package contents, if necessary.
  4. Rename the folder to “filemaker.tmbundle”
  5. Double-click on filemaker.tmbundle
  6. Get to work! TextMate just installed the bundle for you.

For Easy Updates

You can set up the bundle as a Git repository right where TextMate installs it. Here are the Terminal commands:

mkdir -p ~/Library/Application\ Support/TextMate/Bundles
cd ~/Library/Application\ Support/TextMate/Bundles
git clone git://github.com/DonovanChan/filemaker.tmbundle.git "FileMaker.tmbundle"
osascript -e 'tell app "TextMate" to reload bundles'

The TextMate 2 pre-release stores bundles in a different place, however:

mkdir -p ~/Library/Application\ Support/Avian/Bundles
cd ~/Library/Application\ Support/Avian/Bundles
git clone git://github.com/DonovanChan/filemaker.tmbundle.git "FileMaker.tmbundle"
