filemaker.tmbundle by DonovanChan

TextMate Support for FileMaker Pro

This is a bundle for use in TextMate (text editor for Mac OS). It adds functionality that supports development of FileMaker Pro databases.

Become a FileMaker ninja with these tools in your belt!

  • Tab triggers for FileMaker functions
  • Documentation for functions, script steps and error codes
  • Commands
    • Manipulating/generating calculations
    • Manipulating/generating FileMaker clipboard XML
    • Extracting data from the DDR
    • Extracting data from import.log files
  • Syntax highlighting
    • FileMaker
    • FileMaker Clipboard
    • FileMaker Log
    • FileMaker Hash (Generated using #( ) custom function)
  • Code folding
  • Help section

Dependencies

  • TextMate (Available on OS X only. 30-day trial available)
  • Ruby 1.8.7 or newer (should be installed by default on OS X)
  • FileMaker Pro or FileMaker Pro Advanced

Installation

Simple Method

See the installation video.

  1. Install TextMate
  2. Download the files
  3. Rename the folder to “filemaker.tmbundle”
  4. Double-click on filemaker.tmbundle
  5. Get to work! TextMate just installed the bundle for you.

Advanced Method

  1. Clone project into ~/Library/Application Support/TextMate/Bundles
  2. Reload bundles in TextMate
